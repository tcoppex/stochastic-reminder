# StochaPop! - A Stochastic Reminder

_Stochastic Reminder_ is an Android application to schedule random and fixed notifications.

### Features

- Fixed or random scheduling with possible overlapped days,
- Triggered on specific days of the week,
- Specify both the title and long-form multiline message,
- Availables with sound, vibration or muted,
- Exact alarm for non muted notifications.
- Battery preserving scheduling on muted notifications.
